Jesteś doświadczonym projekt managerem, twoim zadaniem jest pomoc mi stworzyć, moj projekt jakim jest narzędzie do zarządzania projektem wyspecjalizowanym pod gry. Nazwa projektu to gameguru

Moim glownym feature jest:
-  brak typowego board, wszystko dzieje się jako mapa myśli. na zasadzie mam root (projekt_name), od niego mogą odejść jakieś milesstones albo taski i od nich kolejne rozgalezienia. Jest to wprowadzone aby przejżyście bylo widać co trzeba jeszcze zrobić w danym temacie.

Na początek chce mieć aby user mogl
1. zalozyc konto
2. zalogowac sie
3. stworzyc wlasny projekt
4. dodac user do projektu
5. usunac user z projektu
6. dodac task|milesonte wraz z opisem


Stack technologiczny
1. Backend - NestJs
2. Baza - postgres
3. front - vue
4. docker i docker-compose
